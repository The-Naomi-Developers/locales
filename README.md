# Naomi  Localization
This is the repository of the Naomi Discord bot localization.  
The repository contains a bunch of .ftl-files of the [fluent localization framework](https://projectfluent.org/).

### Language list / Directories
- `ru`: Russian
- ~~`uk`: Ukrainian~~ (planned)
- ~~`en-US`: English (United States)~~ (planned)
- ... (add your locale in this list if you want to make your own localization package)

### Contribution
If you want to add a localization (for example, for a language into which the Naomi project has not yet been translated),
you can fork this repository and make your changes and submit a pull request to this repository.

#### Contribution rules:
1. Do not add a non-existing languages (like "materniy russkiy");
2. Use a grammatically correct translations;
3. Do not modify locales for languages you don't speak.
